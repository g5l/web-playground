CSS if() Function - Simple POC

⚠️ Note: CSS if() function requires Chrome 137+ with "Experimental Web Platform features" enabled in chrome://flags

1. if() with style() - Theme Detection

Style Condition Example

This box changes color based on the --theme CSS variable.

Current theme: dark

/* if() with style() condition */ .style-example { background: if( style(--theme: dark): #2c3e50; style(--theme: light): #ecf0f1; else: #95a5a6 ); color: if( style(--theme: dark): white; else: black ); }

2. if() with media() - Responsive Design

Media Condition Example

This box is RED on small screens (≤600px) and BLUE on larger screens (>600px).

Resize your browser window to see the change!

/* if() with media() condition */ .media-example { background: if( media(max-width: 600px): #e74c3c; media(min-width: 601px): #3498db; else: #95a5a6 ); padding: if( media(max-width: 600px): 15px; else: 30px ); }

3. if() with supports() - Feature Detection

Supports Condition Example

This box shows:

  • GREEN if CSS Grid is supported
  • ORANGE if only Flexbox is supported
  • RED if neither is supported
/* if() with supports() condition */ .supports-example { background: if( supports(display: grid): #2ecc71; supports(display: flex): #f39c12; else: #e74c3c ); display: if( supports(display: grid): grid; supports(display: flex): flex; else: block ); }
